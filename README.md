# HTML Form

HTML Form adalah fitur dalam HTML yang digunakan untuk mengumpulkan data dari pengguna melalui halaman web. Form memungkinkan pengguna untuk memasukkan informasi seperti nama, email, password, pilihan data, dan input lainnya.

Form berperan sebagai penghubung antara pengguna dan sistem, di mana data yang diinput dapat diproses menggunakan backend atau JavaScript.

## Materi yang Dipelajari
- Struktur dasar HTML Form
- Elemen form seperti input, label, textarea, dan button
- Jenis input (text, email, password, checkbox, radio)
- Atribut penting pada form
- Cara kerja pengiriman data form

## Catatan
HTML Form bukan bagian yang berdiri sendiri. Form biasanya digunakan bersama:
- **CSS** untuk mengatur tampilan form
- **JavaScript** untuk validasi dan interaksi
- **Backend** untuk mengolah data yang dikirim

Memahami HTML Form sangat penting untuk membuat website yang interaktif dan mampu menerima input dari pengguna.
